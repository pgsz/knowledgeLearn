# Stop Doing List

2025-11-04

[你的认知，是财富还是内耗？| 关于拖延的 8 条Stop Doing List](https://sspai.com/post/102316)



#### 第一：别沉溺于空想的美好世界

#### 第二：不要陷入多巴胺的恶性循环

#### 第三：不要奢望拖延有一天会消失

​		接收拖延会一直存在，学会在它出现时，不被它牵着走。处理拖延的正确方式：不是【消灭】拖延，而是去【管理】拖延。

#### 第四：不要一下子就想干把大的

​		慢慢来，会很快。在反复的挫败里，会加深【只有干把大的才能成功】的偏执，陷入越急躁越停滞的怪圈。

#### 第五：不要陷入【完美主义陷阱】

​		最重要的不是完美，也不是完成，而是【开始行动】。

#### 第六：不要频繁切换任务

​		效率的本质不是做更多事，而是把重要的事做好。

#### 第七：不要把【没时间】当借口

​		没时间本质是：你觉得这件事的优先级不高。

#### 第八：不要在阶段性摆烂后攻击自己

​		生命不是永动机，允许自己适当停摆的时刻，是为了走的更远。避免陷入：焦虑 => 更不想做 => 更焦虑 的恶性循环。 



**别人给的，是地图；自己走的，才是路。**

**认知决定你的方向，执行力决定你的终点。**

**真正的成长，不是思想的堆叠，而是行动的积累。**
