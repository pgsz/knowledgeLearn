## 列表如何处理同类数据



列表的定义：

- 列表中可以存放一系列对象
- 存放的对象可以是 数字、字符串 也可以是 列表
- 列表用 **[]** 表示，每个对象称作列表的元素，元素之间用“,”分割
- 可以在定义之后进行修改，可以用索引访问

列表的创建：

- 使用一对方括号来表示空列表: `[]`
- 使用列表推导式: `[x for x in iterable]`
- 使用类型的构造器: `list()` 或 `list(iterable)`
