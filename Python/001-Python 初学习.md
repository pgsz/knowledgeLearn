## 初步学习

文档：[https://docs.python.org/zh-cn/3/](https://docs.python.org/zh-cn/3/)

Python 是一种 **高级、解释性、面向对象的动态强类型**语言。

JavaScript 是一种 **高级、解释性、弱类型、基于原型的动态**脚本语言。

1. 非交付执行 Python 脚本：  py | python text.py 。
2. 交互执行 Python 脚本： py | Python -i test.py 。 Python 解释器执行用户命令之后，用户可以继续输入指令。
